# Currency Exchange API

Простое REST API для получения актуальных курсов валют (USD и EUR)

## 📋 Описание

Этот проект представляет собой микросервис на FastAPI, который получает данные о курсах валют с официального API ЦБ РФ и предоставляет их через удобный REST эндпоинт.

## 🚀 Возможности

- Получение актуального курса USD (Доллар США)
- Получение актуального курса EUR (Евро)
- Автоматическое округление значений до двух знаков после запятой
- Временная метка последнего обновления данных
- Обработка ошибок при недоступности внешнего API

## 🛠 Технологии

- **FastAPI** - современный веб-фреймворк для создания API
- **Requests** - библиотека для HTTP-запросов
- **python-dotenv** - управление переменными окружения

## 📡 API Эндпоинты

### `GET /rates`

Возвращает актуальные курсы валют USD и EUR от ЦБ РФ.

#### 📥 Пример успешного ответа (200 OK)

```json
{
  "USD": 92.50,
  "EUR": 98.75,
  "timestamp": "2024-01-15T12:00:00+03:00"
}
```
| Поле | Тип | Описание |
|------|-----|----------|
| USD | float | Курс доллара США в рублях |
| EUR | float | Курс евро в рублях |
| timestamp | string | Время последнего обновления данных |


####Пример ответа с ошибкой:

```json
{
  "error": "Ошибка при получении данных: ..."
}
```
